# Petish

Petish는 반려동물 산책메이트 & 커뮤니티입니다.

## Project Setup

- **Language**: Dart
- **FE Framework**: Flutter
- **BE Framework**: Spring Boot
- **Database**: MySQL
- **Cloud Service/Platform**: Firebase
- **IDE**: IntelliJ IDEA, Android Studio

## Requirements

- Flutter SDK Version: >=3.1.5 <4.0.0 (As specified in `pubspec.yaml`)
